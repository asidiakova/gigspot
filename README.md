# About GigSpot

GigSpot allows you to find the latest info about the upcoming underground concerts and gigs. Be an organizer or an attendee, create 
and join the events, and never miss a good moshpit!

## Features

- **For Organizers**: Create and manage events, upload flyers, track attendance, build your follower base
- **For Attendees**: Browse upcoming events, mark attendance, follow your favorite organizers, filter by city, and keep track of the upcoming events
